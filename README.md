# PythonAssignment_4

Task 1: Read a File and Handle Errors

This program opens a file named sample.txt, reads its content,and prints it line by line. It includes a try-except block to gracefully handle cases where the file might be missing.

Key Features:
->Uses with open() for safe file handling.
->Exception handling for FileNotFoundError.

Task 2: Write and Append Data to a File

This program interacts with the user to collect data and perform the following:
->Creates output.txt and writes the first set of input.
->Appends a second set of user input to the same file.
->Reads the entire file back to display the final result.
